# DKS Liberia - Official Website

A production-quality website for dksliberia.org, built with Next.js and TailwindCSS.

## Tech Stack

- Next.js (Pages Router)
- TailwindCSS
- Framer Motion
- Swiper.js
- React CountUp
- Chart.js
- TypeScript

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for deployment on Vercel. Simply connect your repository to Vercel and deploy.

## Project Structure

- `/pages` - Next.js pages and API routes
- `/components` - Reusable React components
- `/data` - JSON data files for content
- `/public` - Static assets (images, etc.)
- `/styles` - Global styles

## Notes

- All dynamic data is currently loaded from local JSON files in the `/data` folder
- API routes include placeholder comments for backend integration
- The site is fully responsive and SEO-optimized
